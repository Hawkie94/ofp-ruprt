Version history
Version 3.1.0
Release date : 02/26/2005

1) Fixed an error precluding application start. This error appears when the user changed the the first keywords' letter casing to lower.
2) Fixed bad date and time displaying of de/compressed files in PBO Manager. Now the files are of the correct original date and time.
3) Now it's possible to associate *.PBO files with PBO Manager through the OFPMDE options panel. Command line decompression support is implemented as well.
4) Fixed the Windows system registry cleaning when OFPMDE is uninstalled. In the previous version some of the OFPMDE registry records remain in registry after uninstallation.
5) Hardly improved registry records management.


Version 3.0.0. - Upgrade
Release date : 12/24/2004

1) Completely rebuild for use with Compact NET.Framework
2) Totaly changed an accelerated the syntax highlighting. You can use now files of an unrestricted size.
3) Syntax highlighting works with all of OFP file types.
4) Code Checker works with all OFP file types.
5) Error message panel (bottom) added.
6) Function for code informations gathering added (works with all OFP file types):
    - SQS -> Labels list
    - EXT, CPP, SQM -> Classes and constants structure and hierarchy
    - CSV -> String sets names list
7) Predefined list of EXT and CPP constants added to the right panel
8) Many UI options added.
9) File associations process improved.
10) Function Find &Replace improved. Now it's possible to search in closed project files.
11) Undo and Redo operations improved.
12) During creation of new project it's now possible to select which files should be added into a mission folder.
13) Function "File information header inserting" added. These headers are predefined texts which will be automatically inserted in the top of an each new file. These texts can be also changed by editing fileinfo.xml file installed in the Data\ subfolder of the application folder.
14) Projects panel's local menu improved. Now it's possible to directly extract PBO files, decode BIN files or pack mission folder.
15) Application has been localized to ENG.
16) For ENG and G locaclizations a simplified context help window has been implemented. In this window are now displayed descriptions of scripting command aquired directly from the official BIS help instead of the Czech OFPEditorHelp.
17) PBO Manager tool added. This tool can pack, unpack PBO files (optionaly using compression) and decode BINary files.
18) Debug Console Manager improved, Debug Console localized to ENG.
19) Color Calculator tool improved.
20) Code Snippits tool removed.


Version 2.2.0 - Update
Release date : 02/17/2004
1) Syntax highlighting algorithm improved
2) Quick Keywords Menu now automaticaly disapears if no keyword is found and won't be shown if the caret is on a comment line.
3) It's possible now to set up the size of the first keyword's letter (i.e. setDammage or SetDammage)
4) Automatically inserted comments in a file header removed.
5) No empty file on application startup any more.
6) Font size in Pojects panel scaled down.
7) Find & Replace now can search for whole words only and replace all found words at once.
8) Background color can be set up now.
9) Checking of correct count of parenthesis and quotes added.
10) File associations bug fixed.
11) Copy/Paste bug fixed.
