OFP MDE
Since the time I had fallen to the Operation Flashpoint and mission making, I have been missing some tools which would do my work on mission developing easier. First I didn't understand the scripting commands, so I wrote the OFPEditorHelp (czech only - I hope you all know it ). Then I had missed some kind of testing tool, so the Debug Console came into existence. And finally when I had felt almost satisfied... then Didymos showed me his script file for CSLA II. Campaing Intro with a size of 62 kb and 2417 lines. At this moment I awoke that the common Windows Notepad isn't the kind of "cool" editor. I think you agree with me that such a script isn't good readable whent it's written in one color. And this was the moment I started to think of some kind of script editor which would be a little more handy.

My idea was to build something like Visual Basic Code Editor. The main requests were following :
a) to distinguish particular parts of code (keywords, comments, double-quoted texts),
b) to open more than one file in one application with an easy switching between them,
c) to dynamiclly offer keywords in an ordered list,
d) to check for some types of often errors such like missing parenthesis etc.,
e) to immediatelly show a help for a concrete keyword.

When I had everything I needed to know I started to develope the OFP Mission Development Environment. After that the first version had been released, I had gotten some feedback from users and also the VB MCP(Microsoft Certified Professional) cerificate (I have to air myself ) I started to build an upgrade in more robust and incrediblly extensive development environment which is the Visual Studio .NET 2003. Therefor this upgrade already includes real "developers" features instead of the previous version which has been just such a kind of a plain and slow syntax highlighter.

The development of the first version took me approximatelly 6 months and this upgrade almost 1 year so I hope that my work was not useless and the OFP MDE will be good helper for making missions of your own.
