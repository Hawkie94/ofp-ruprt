Color Calculator
A handy tool for addon and dialog window makers. It can calculate all possible values of a given color.
In the computer world colors are expressed in numbers. In case you would like to add a specific background color to a some of your dialog window of your mission you have to express it in numbers. For this reason I made the Color Calculator.
It seems to be an useless tool but believe me.... this tool made my work easier and I hope that it will do the same for you :o))
