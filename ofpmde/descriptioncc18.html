Quick Keywords Menu
This function provides users with a dynamic local menu which includes an alphabeticaly ordered list of all possible scripting commands with an ability to insert a selected keyword into an opened document.

The function behaves depending on a current file type and a caret position. Depending on the setting the menu will be shown when given written count of letters match the same first count of some keyword letters (default count is 2).

If you write 2 letters down and some match will be found then the local menu will be shown and first selected keyword will begin with these two letters. It also means that if no match has been found then no loacal menu appears. In case you write "zk" no menu appears because of no keyword begins with "zk". On the other hand if you write :
Written text Keywords in the local rolling menu
c Nothing
cl ClearMagazineCargo
clo CloseDialog
cloz Menu disapears


When the menu is shown you can go trough its items using Arrow keys or ScrollBar and insert requested keyword by pressing Tab or Enter key or by double_clicking. Menu disapears when you use the Escape key.
