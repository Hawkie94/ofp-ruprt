Context Help
This function offers users an immediate help for concrete scripting command.

Each mission maker needs ever and again to look into the scripting command help to find out an exact syntax or command description. For this reason my OFPEditorHelp (czech only) or the official Scripting Commands References manual serve. If you use such a help you must first select the appropriate command and then to click on it. Context help does this annoying work for you.

If you right-cklick in an opened document on a keyword then a local menu with one item "Explain command : " + keyword will be shown. When the Context help finds this keyword then opens immediate window for you with the keyword description. If the Context Help finds no such a keyword then the local menu item shows just "Explain command : Unknown" and offers you to open the entire help.
