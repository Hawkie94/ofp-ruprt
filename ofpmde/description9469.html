Code Checker
This function cooperates with the Syntax Highlighter and its task is to check a document against errors depending on a file type. If any error is found (i.e. not matching count of parenthesis in SQS or wrong class structure in EXT, bad CSV header in CSV) the error is underlined and its description is shown in the bottom panel. If you double-click on this description the caret will be moved at an appropriate position.

In addition after the code is cheked against any error important code info will be shown in the right panel. What kind of info depends on a file type again. For SQS it is a Labels list, for EXT, SQM, CPP it is a classes structure and hierarchy including constants definitions, for CSV it is a string sets names list (when you double-click on an item in this info the caret will be moved at an appropriate position).
