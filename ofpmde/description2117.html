Debug Console
A tool for a mission debugging. In the mission run-time you can get informations about objects positions, variables' values etc. It is even possible to enter any valid expression and get its value or execute any valid command or function directly from running mission. The Debug Console has its own Manager for installing to and uninstalling from missions.It is strongly recommended to always use this manager. Keep in mind that by installing Debug Console into your mission the description.ext and stringtable.csv files will be changed, but don't worry about this because the manager always makes a backup copy of these files for you.
