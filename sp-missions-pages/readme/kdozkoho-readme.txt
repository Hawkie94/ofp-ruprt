Tato mise "Kdo z koho." vyžaduje pro bezproblémový chod následující uzivatelske parametry:

Operace FlashPoint verse 1.30 a vyšší

a

nasledujici nainstalovane AddOny :

1)
Název AddOn : H&K PSG-1 (neoficialni AddOn)
Název Souboru : psg.pbo
Download mozny z http://www.operationflashpoint.d2.cz/ sekce AddOny -> Zbraně

2)
Název AddOn : H&K PSG-1 Silenced (neoficialni modifikovaný AddOn)
Poznámka : Je třeba nainstalovat PŘILOŽENÝ AddOn "psgs.pbo", což je
	   modifikovaný AddOn od Devil downloadovatelný ze stránek
	   http://www.operationflashpoint.d2.cz/
Název Souboru : psgs.pbo
AddOn je přibalen v downloadovaném souboru "kdozkoho.zip"

3)
Název AddOn : Civilní sanitka & doktorka  (neoficialni AddOn)
Název Souboru : bpg_add07.pbo
Download mozny z http://www.operationflashpoint.d2.cz/ sekce AddOny -> Vozidla

4)
Název AddOn : Pridavky 3 (Oficialni balicek AddOns)
Název souboru : OFP_AddOns Pack 3_CZE_PLN.exe
Download mozny z http://www.bistudio.cz/flashpoint/


Instalace :
Dekomprimujte downloadovaný soubor kdozkoho.zip a zkopirujte soubor kdozkoho.pbo
do adresare (slozky) " ..\OperationFlashPoint\missions\"a nasledne spuste z prostredi
hry stejne jako jakoukoli jinou "Jednotlivou misi".
